# 一起来造DIP轮子吧

DIP即 Digital Image Processing。顾名思义，让我们一起来从基础开始构建数字图像处理的函数库吧。

## 说明
### 参考资料
1. Digital Image Processing 3e
2. Digital Image Processing using Matlab
3. opencv documents

### 轮子内容
1. DIP一书中所涉及道德方法和操作
2. 自己感兴趣的图像处理的相关操作

### 技术标准
1. 可以使用python的各种科学库，但不包括CV2
2. 给出合适的文档（English）
3. 除了 读入、显示、储存图像外，不能使用python的内建图像库
